# ⚡ Goro Energy - NVIDIA Inspired E-commerce

> **Goro Energy** é uma plataforma de e-commerce de alta performance, focada em produtos de energia e tecnologia, com uma interface industrial e minimalista inspirada no design system da NVIDIA.

---

## 🛠️ Tecnologias

O projeto utiliza uma stack moderna para garantir performance e escalabilidade:

### Frontend
- **React 19** - UI Reativa e moderna.
- **Vite** - Build tool ultra-rápida.
- **Tailwind CSS v4** - Estilização baseada em utilitários e design tokens.
- **Framer Motion** - Animações suaves e interações premium.
- **Zustand** - Gerenciamento de estado leve e eficiente.
- **Lucide React** - Ícones consistentes e elegantes.

### Backend
- **Express.js** - Servidor RESTful robusto.
- **SQLite (Better-SQLite3)** - Banco de dados local rápido e confiável.
- **JWT & BCrypt** - Segurança e autenticação de ponta.
- **Concurrently** - Execução simultânea de frontend e backend.

---

## ✨ Funcionalidades

- 🔐 **Autenticação Completa**: Cadastro e Login com persistência via JWT.
- 🛒 **Catálogo Dinâmico**: Visualização de produtos com design industrial.
- 💳 **Checkout Otimizado**: Experiência de compra fluida e segura.
- 📱 **Design Responsivo**: Totalmente adaptado para dispositivos móveis e desktops.
- 🎨 **NVIDIA Tech Aesthetic**: Tema dark de alto contraste com acentos em Verde NVIDIA (`#76b900`).

## 🚀 Como Iniciar

### Pré-requisitos
- [Node.js](https://nodejs.org/) instalado.

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/KauaViReis/goroDaMansao.git
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor de desenvolvimento (Frontend + Backend):
```bash
npm run dev
```

O frontend estará disponível em `http://localhost:5173` e o backend em `http://localhost:3001`.
